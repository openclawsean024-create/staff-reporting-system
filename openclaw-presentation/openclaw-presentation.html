安裝設定

快速開始

安裝步驟
1
安裝 OpenClaw
npm install -g openclaw
2
初始化
openclaw init
3
連接平台
設定 Token
4
啟動使用
openclaw start

系統需求

  • Node.js 18+
  • npm 或 yarn
  • 4GB+ RAM
  • 支援 Docker

部署選項

  • 本地部署（開發者）
  • Docker Compose（伺服器）
  • 雲端 VPS
  • Cloudflare Tunnel
